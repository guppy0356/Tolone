# Family Todo

A shared family todo app where all members can see everyone's todos, but can only complete or delete their own.

## User Switching

The current user is determined by the `currentUser` cookie (defaults to `Papa`).

To switch users:

1. Open DevTools > Application > Cookies
2. Change the `currentUser` value to `Papa`, `Mama`, `Taro`, or `Hanako`
3. Reload the page
